# Go Svelte kit Example

Make Svelte SSR run in Golang

### Requiment

- Nodejs, pnpm
- Golang 1.16 or above

### How to start?

```
make build-client # build svelte kit
make run-server #
```

Open website with localhost:3000 for test render sveltekit with Go
